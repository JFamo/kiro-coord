# kiro-coord

A web-based tool to coordinate multiple Kiro CLI agents locally. Manage multiple chat sessions with different agents through a clean interface.

## Features

- Create and manage multiple Kiro agent sessions
- Switch between active sessions
- Real-time chat interface for each agent
- Close sessions when done

## Tech Stack

- Backend: FastAPI with WebSocket support
- Frontend: React with Material-UI
- Process Management: Python asyncio for Kiro CLI instances
